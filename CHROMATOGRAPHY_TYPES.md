# 色谱类型与归一化基准

## 概述

根据不同色谱类型的实际废液产生量，系统使用不同的归一化基准质量进行评分计算。

## 色谱类型对照表

| 色谱类型 | 英文缩写 | 废液基准质量 | 说明 |
|---------|---------|------------|------|
| 合相色谱 | UPCC | 4g | 超临界流体色谱，废液最少 |
| 超高效液相 | UPLC | 4g | 超高效液相色谱，废液很少 |
| 常规HPLC (LC-MS) | HPLC-MS | 10g | 液质联用色谱 |
| 常规HPLC (UV) | HPLC-UV | 45g | 紫外检测器色谱（默认） |
| 半制备HPLC | Semi-prep | 250g | 半制备级液相色谱，废液最多 |

## 归一化公式

```
小因子得分 = min(100, (Σ(质量 × 因子) / 基准质量) × 100)
```

### 示例

假设使用10g试剂，因子值为1.0：

- **UPCC/UPLC (4g基准)**：(10 × 1.0 / 4) × 100 = **100分**（达到上限）
- **HPLC-MS (10g基准)**：(10 × 1.0 / 10) × 100 = **100分**
- **HPLC-UV (45g基准)**：(10 × 1.0 / 45) × 100 = **22.2分**
- **Semi-prep (250g基准)**：(10 × 1.0 / 250) × 100 = **4分**

## 使用建议

1. **根据实际仪器选择**：请根据实验室实际使用的色谱类型进行选择
2. **基准的意义**：基准质量代表该色谱类型的典型废液产生量
3. **评分的相对性**：相同的试剂用量，在不同色谱类型下得分不同
