# 用户手册

## 安装指南

### Windows系统

1. 下载安装包 `HPLC-Green-Chemistry-Setup-1.0.0.exe`
2. 双击运行安装程序
3. 按照提示完成安装
4. 从开始菜单或桌面快捷方式启动应用

### macOS系统

1. 下载 `HPLC-Green-Chemistry-1.0.0.dmg`
2. 打开dmg文件
3. 将应用拖到Applications文件夹
4. 从启动台运行应用

### Linux系统

**AppImage方式**
```bash
chmod +x HPLC-Green-Chemistry-1.0.0.AppImage
./HPLC-Green-Chemistry-1.0.0.AppImage
```

**Deb包方式**
```bash
sudo dpkg -i hplc-green-chemistry_1.0.0_amd64.deb
```

## 功能使用指南

### 1. 绿色化学评估

#### 步骤：
1. 点击左侧菜单"绿色化学评估"
2. 选择溶剂A和溶剂B
3. 输入溶剂A的比例（0-1之间）
4. 输入总体积（mL）
5. 点击"计算绿色化学评分"按钮
6. 查看评估结果

#### 评分说明：
- **整体绿色评分**: 0-100，越高越环保
  - 90-100: 优秀
  - 70-89: 良好
  - 50-69: 中等
  - <50: 需要改进

- **危险性评分**: 溶剂的危险程度（满分10）
- **环境影响评分**: 对环境的影响程度（满分10）
- **健康安全评分**: 对健康的影响（满分10）
- **可回收性评分**: 溶剂的回收难度（满分10）

#### 支持的溶剂：
- 水
- 甲醇
- 乙腈
- 四氢呋喃
- 乙酸乙酯
- 正己烷
- 异丙醇

### 2. 色谱图分析

#### 步骤：
1. 点击左侧菜单"色谱图分析"
2. 输入保留时间数据（用逗号分隔）
   - 例如：2.5, 3.8, 5.2, 7.1
3. 输入对应的峰面积数据（用逗号分隔）
   - 例如：1200, 8500, 450, 300
4. 点击"分析色谱图"按钮
5. 查看分析结果和可视化图表

#### 结果解读：
- **峰数量**: 检测到的色谱峰总数
- **主峰保留时间**: 面积最大峰的保留时间
- **纯度**: 主峰占总面积的百分比
- **平均分辨率**: 相邻峰之间的平均间隔

### 3. 分析记录管理

#### 功能：
- 查看所有历史分析记录
- 筛选和排序记录
- 查看每条记录的绿色评分
- 导出分析报告

#### 操作：
1. 点击左侧菜单"分析记录"
2. 浏览所有分析记录
3. 点击"刷新"按钮更新数据
4. 使用分页功能浏览更多记录

## 最佳实践

### 溶剂选择建议

1. **优先使用绿色溶剂**
   - 水（评分最高）
   - 乙酸乙酯
   - 异丙醇

2. **避免使用高危溶剂**
   - 四氢呋喃（可形成过氧化物）
   - 正己烷（高环境影响）

3. **优化溶剂比例**
   - 增加水的比例
   - 减少有机溶剂用量
   - 测试不同比例以找到最佳方案

### HPLC方法优化

1. **流速优化**
   - 较低流速减少溶剂消耗
   - 平衡分析时间和溶剂用量

2. **温度控制**
   - 室温操作最环保
   - 避免极端温度

3. **色谱柱选择**
   - 使用高效色谱柱缩短分析时间
   - 减少溶剂消耗

## 故障排除

### 问题1：应用无法启动
**解决方案：**
- 检查系统要求是否满足
- 重新安装应用
- 查看系统日志

### 问题2：后端连接失败
**解决方案：**
- 检查防火墙设置
- 确保端口8000未被占用
- 重启应用

### 问题3：数据无法保存
**解决方案：**
- 检查磁盘空间
- 确保有写入权限
- 清理旧数据

### 问题4：计算结果异常
**解决方案：**
- 检查输入数据格式
- 确保数值在合理范围内
- 重新输入数据

## 数据管理

### 数据存储位置

**Windows**: `C:\Users\<用户名>\AppData\Local\HPLC_Green_Chemistry\data`
**macOS**: `~/Library/Application Support/HPLC_Green_Chemistry/data`
**Linux**: `~/.config/HPLC_Green_Chemistry/data`

### 备份数据

1. 找到数据存储目录
2. 复制 `hplc_analysis.db` 文件
3. 保存到安全位置

### 恢复数据

1. 关闭应用
2. 将备份的数据库文件复制回数据目录
3. 重启应用
